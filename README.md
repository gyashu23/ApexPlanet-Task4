# ApexPlanet-Task4
A modern and responsive Personal Portfolio Website built using HTML, CSS, and JavaScript.

## Features
- 👤 Responsive personal portfolio
- 🏠 Home, About, Skills, Projects, and Contact sections
- 💻 Project showcase with navigation to individual projects
- 📱 Fully responsive design for desktop and mobile devices
- 🎨 Clean, modern, and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Projects Included
- ⌚ Smart Watch E-Commerce Website
- 📝 Task Planner Application

## How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
